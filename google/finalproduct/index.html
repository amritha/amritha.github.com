Final Wireframes

After sketching out various options, I created wireframes for the concept that I decided upon. I named the product KitchenMate.

1. KitchenMate's Location

KitchenMate is a tablet-sized device built into the kitchen's Island area between the two placemats for eating.

2. Home Screen

Mom can see this week's and next week's grocery lists on the KitchenMate home screen. The upcoming week's list will become this week's list at 12 AM every Sunday.

  • 2.1 Items that don't recur float to the top of the list for higher emphasis.
  • 2.2 Recurring items are visually indicated with a different background color.
  • 2.3 User can check off items by tapping in the checkbox area upon which the item is added to the top of the list of checked off items. Tapping on the checkbox again can uncheck the item and return it back to the pending items section.
  • 2.4 User can toggle between this weeks grocery list and next weekâ€™s.

3. Add Item

Mom can add multiple items to the this week's or next week's grocery list by hitting the giant + button.

  • 3.1 User can type in item name.
  • 3.2 User can choose to indicate that this item recurs weekly by tapping on the checkbox.
  • 3.3 Tapping on "Add Another Item" allows the user to add multiple items in one screen. Tapping it causes another set of input fields to appear below the existing one.
  • 3.4 Once the user starts typing, they can clear the contents of the field by tapping the "x."
  • 3.5 Hitting "Done" adds these items to the list that the user tapped the "+" from.

4. Edit Item

Mom wants to make sure that Milk is no longer a recurring item. She can do this by tapping on the Milk item in her list to take her to the edit screen.

  • 4.1 Once user taps on an item, they are taken to the edit screen. Here they can change the name of the item or alter whether it is a recurring item or not.
  • 4.2 User can also choose to delete this item upon which they are taken back to the list that they came from.

5. Synced with Phone

Dad decides to glance over the list in the kitchen before he leaves home. When he's about to check out at the grocery store, he quickly takes a look at the list again on the KitchenMate mobile app.

6. Leaving Notes

Mom wants to leave a note for Kid that reminds Kid to take her antibiotics. Once she leaves the note, the indicator flashes to get Kid's attention.

  • 6.1 Tapping on the "Post a Note" button on the Home screen allows the user to input a note.
  • 6.2 On hitting post, the note is posted as an overlay over the grocery list. The light indicator will flash to indicate there is an unread note.
  • 6.3 Upon hitting ok, the note disappears and the indicator stops flashing.
  • Note: Notes feature is not available on KitchenMate mobile app because KitchenMate notes are intended to be consumed in the kitchen.

Future Considerations

  • It would be great to test this product out with many other families to see if this fulfills any of their needs.
  • Will families want more robust note management? Or is a transient announcement enough?
  • Will users have any confusion keeping up with whether they are on this week's or the upcoming's week list?
  • How will kitchen behavior change in this family because of this device?
  • Will any other pain points emerge now that grocery list management has become slightly easier?
